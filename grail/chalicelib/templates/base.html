About the app

Grail Publishing is a simple image publishing service, built on AWS Lambda + S3. Images are posted via the command line. Your image is ran through the AWS Rekognition service, which identifies interesting objects in the image itself. Next, we search an Elasticsearch index of XKCD posts, for the labels that Rekognition found. Finally, we create an HTML page to contain your image, as well as the XKCD post most relevant to your image.
